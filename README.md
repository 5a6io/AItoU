# AltoU Server
## 개발 환경
- Spring Boot 3.1.3, Java 17, MySQL
## API
- 회원 등록
- 출석 체크 및 전체 조회
- 사용자와 AR 간의 음성 응답
- 발음 평가 결과
## 사용 라이브러리
- Google Speech To Text
- Google Text To Speech
- Google Cloud Storage
- Azure AI
